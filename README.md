# 🚀 TechLogic Studio AI

TechLogic Studio AI is a desktop application built with Python and PySide6 for AI-powered content creation.

## Features

- 🎬 AI Video Generator
- 🖼 AI Image Generator
- 🎨 Blender Integration
- 📁 Asset Manager
- 🎙 Voice Generator
- 🤖 Prompt Library
- 📦 Project Manager

## Tech Stack

- Python 3.11
- PySide6
- Git
- GitHub
